# Python Visual Building Controller(RTU) with ISO 16484-5 BACnet Communication Protocol

**Project Design Purpose** : In this project, I extend the Python-Based virtual RTU simulator system (which interfaced to SCADA system via Modbus-TCP, Siemens-S7Comm, IEC 60870-5-104 and IEC 62541 OPC-UA-TCP)  by adding the support for ISO 16484-5 BACnet protocol.
