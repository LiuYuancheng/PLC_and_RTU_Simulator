# Python Virtual RTU/IIoT Simulator with IEC-20922 MQTT Communication Protocol

**Project Design Purpose** :  In this follow-up project, we extend the previous Python-based virtual PLC/RTU simulator project (which interfaced to SCADA systems via Modbus-TCP and S7Comm, related link:  https://www.linkedin.com/pulse/python-virtual-plc-rtu-simulator-yuancheng-liu-elkgc)  by adding the support for IEC-20922 Message Queuing Telemetry Transport (MQTT) protocol.
